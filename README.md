## 📌 项目简介

本项目是一个基于 **Flask + SQLite + 原生 HTML/CSS/JS** 开发的**全功能在线通讯录系统**，支持联系人信息的增删改查、收藏、CSV 导入导出、关键词搜索、深色/浅色主题切换等核心功能。

项目严格遵循**极限编程（XP）** 的协作开发模式，采用结对编程、持续集成、小步快跑的开发节奏，最终成功部署于阿里云 ECS 服务器，实现 7×24 小时在线访问。


## ✨ 核心功能

- ✅ **联系人管理**  
  - 添加姓名、地址、多种联系方式（电话、邮箱等）
  - 编辑与删除已有联系人
- ⭐ **收藏功能**  
  - 一键标记/取消收藏重要联系人
  - 独立展示“收藏夹”列表
- 📥 **CSV 批量导入**  
  - 支持标准格式 CSV 文件上传（列：姓名、电话、邮箱、住址）
  - 自动跳过空行，容错处理
- 📤 **CSV 导出**  
  - 一键导出当前所有联系人为 `.csv` 文件
- 🔍 **实时搜索**  
  - 按姓名模糊匹配，动态过滤列表
- 🌙 **主题切换**  
  - 支持深色/浅色模式，提升用户体验
- ☁️ **云端部署**  
  - 部署于阿里云 Ubuntu 24.04 服务器
  - 公网 IP 直接访问，无需本地运行
